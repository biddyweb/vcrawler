# vcrawler

video crawler

---------

### 目标

下载bilibili的视频

### 运行环境

- `pypy` / `python2.7`
- 使用`axel`下载 / 可自行改为`wget`

----------

### 使用

```bash
# usage
./get_video.py (av | url)

#./get_video.py av26894
#./get_video.py http://www.bilibili.tv/video/av26894/
#./get_video.py http://www.bilibili.tv/video/av26894/index_2.html
```

----------
